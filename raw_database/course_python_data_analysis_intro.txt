課程概說：
本課程為資料分析初學者設計，以 Python 為工具，涵蓋資料清洗、探索性分析、視覺化與簡易模型建立。學生將實際操作 pandas、matplotlib、seaborn 等套件，學會如何處理結構化資料並生成有意義的商業洞察。

16 週課程內容提要：

Python 基礎語法與環境建置
資料結構與檔案處理（csv, json）
pandas 資料框架操作
資料清洗與缺失值處理
資料過濾、排序與分組
描述統計與分佈觀察
資料視覺化（matplotlib/seaborn）
時間序列與類別資料處理
合併與彙總分析
文字資料的簡易分析
異常值偵測與商業洞察
簡易迴歸模型實作
使用 sklearn 做簡易預測模型
實務案例：行銷活動成效分析
專案開發與報告撰寫技巧
小組專題發表與成果展示
